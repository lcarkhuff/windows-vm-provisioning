# Window VM Provisioning
Infrastructure-as-Code project for provisioning new Windows Server VM's using:
  -Terraform (vSphere provisioning)
  -Ansible (Configuration management)

## Purpose
This repository provides a repeatable, automated process for building Windows Server VM's in VMWare

## Structure
terraform/  -> VM provisioning
ansible/    -> Post-build configuration
docs/       -> Design notes and documentation
